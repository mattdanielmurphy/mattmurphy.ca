# Physics Lab Web Synchronization

This folder contains the synchronization scripts that import Physics 12 laboratory data from the local academic workspace (`School`) into the personal portfolio site.

---

## 🛠️ Sync Pipeline

### `copy-physics-labs.js`
This script acts as the automated import bridge. It is designed to be executed stand-alone or auto-triggered instantly whenever `pdf-exporter.js` compiles a lab in the `School` directory.

#### What it does:
1. **Recursive Scan**: Scans `School/physics-12/` for any `.preview.html` files generated by the PDF exporter.
2. **Slug Calculation**: Normalizes filenames into clean, URL-safe slugs (e.g. `ph12-u1-free-fall-lab-2d-projectiles`).
3. **Image Harvesting**: Parses each HTML document for local `<img>` tags. It copies the source images from the `School` lab directories to `public/physics-labs/images/`, renaming them to include the parent slug to guarantee zero filename collisions.
4. **HTML Formatting & Insertion**:
   - Removes absolute/local `<base>` paths so references load relative to the site route.
   - Injects a styled back-button block (`← Back to Physics Labs`) at the top of the body for seamless navigation within iframe embeds.
   - Saves the updated HTML files under `public/physics-labs/`.
5. **Manifest Generation**: Generates `public/physics-labs/manifest.json` containing metadata (slugs, parsed titles, and extracted unit groupings).
6. **Obsolete Cleanup**: Compares active labs against files on the web server and automatically unlinks/deletes any older, deleted, or renamed HTML reports.

---

## 🌐 Site Directory Rendering (`/physics`)

- **Routing**: Next.js serves the directories statically using the generated manifest.
- **Card Display Cleanups**: In `src/pages/physics/index.js`, card titles are styled and processed on-the-fly to hide rigid course codes while preserving the underlying metadata:
  ```javascript
  // Truncates "PH12 U1 Free-Fall Lab" to "Free-Fall Lab"
  lab.title.replace(/^PH\d+\s+U\d+[A-Z]?[:\s]+/, '')
  ```
- **Print media considerations**: The source PDFs contain web recommendation links. These links are automatically hidden (`display: none;`) inside the website's pages using standard CSS styles in `pdf-exporter.js` but will render cleanly on print/PDF viewports.
